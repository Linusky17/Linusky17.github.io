Web Apps

UVA Study Buddy Finder

  • The UVA Study Buddy Finder is a website, specifically a web app, designed to assist UVA students to find study peers according to their need. It allows users to set up their account with their interests, course schedule, what they need help with, as well as other information related to finding study peers, and it generates study buddies and study groups dynamically based upon similar learning needs and interests.

    Built primarily using Django Framework, Python and Bootstrap. Deployed using Heroku and PostgreSQL (originally using SQLite).

Tunescope (University of Virginia, Make to Learn Lab)

  • Project is still under development!

    A computer science online learning environment with three main purposes: (1) teach users to create music and other things using block programming, (2) allow users to visualize the amplitudes and frequencies of musical notes, and (3) collect usage data for use in training artificial intelligence to offer music synthesis suggestions

    Serve as a software engineer of the team. Contribute to all features involving the project using HTML, CSS, JavaScript and Node.JS.
